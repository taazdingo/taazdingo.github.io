Boss Order System

The Boss Order system is one of the key elements in the Level Design

Each level begins with the Boss ordering an initial set of fruits (X).

Each order has a fixed timer running before which the corresponding fruit must be delivered to the Boss.

During the course of a level, the Boss orders more fruit. The total number of fruit the Boss orders during an entire level (Y) is increased each level.

Each level may contain only a few, or all types of plants in the game. The number of types of plants in each level (Z) is varied for a challenging gameplay.

These parameters X,Y and Z contribute to a level's complexity.

The Boss can switch between three states. Idle, Angry and Punishing

Idle State: Boss placed order and is waiting.

Angry State: One order has missed. (Order time elapsed before fruit could be thrown)

Punishing State: Another consecutive order missed.
